# Тестовое задание для [Outside Digital](https://outsi.de/)

[Ссылка на деплой на Github-pages](https://melnikovaleksei.github.io/test-outside-kodix/)

### Установка:

```
git clone https://github.com/MelnikovAleksei/test-outside-kodix.git

cd test-outside-kodix

npm install

npm start
```

### Используемые технологии:

* Reactjs
* CSS
* HTML

### Используемые библиотеки:

* [react-uuid](https://www.npmjs.com/package/react-uuid)
* [gh-pages](https://www.npmjs.com/package/gh-pages)
